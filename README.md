# etok.zo.space

> My personal Zo Space — [https://etok.zo.space](https://etok.zo.space)

## Overview

A living, breathing personal site built on [Zo Computer](https://zo-computer.cello.so/fFG5xDTfXhY). Features a subtle Conway's Game of Life animation in the hero, live PST clock, and a dark green aesthetic.

## Routes

| Route | Description |
|-------|-------------|
| `/` | Homepage — bio, links, projects, currently |

## Development

Zo Space routes are managed via the Zo dashboard. This repo stores snapshots of route code for version history.
